# Анализ рынка недвижимости Санкт-Петербурга

## Описание проекта

Исследование архивных данных сервиса Яндекс.Недвижимость о продаже квартир в Санкт-Петербурге и соседних населённых пунктах за несколько лет. Цель проекта — определить параметры для построения автоматизированной системы оценки рыночной стоимости объектов недвижимости и выявления аномалий.

## Технологии

- **Python 3.8+**
- **Pandas** - обработка и анализ данных
- **Matplotlib/Seaborn** - визуализация
- **Jupyter Notebook** - интерактивный анализ
- **NumPy** - математические вычисления
