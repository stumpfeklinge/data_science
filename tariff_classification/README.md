# Рекомендация тарифов

## Описание проекта
Цель проекта — построить модель машинного обучения, которая будет рекомендовать пользователю подходящий тариф мобильной связи на основе его поведения.

Модель должна определить, какой тариф лучше подходит пользователю:
- **0 — Smart**
- **1 — Ultra**

Минимальное требование к качеству модели: **accuracy ≥ 0.75**.

---

## Данные

Используется датасет `users_behavior.csv`.

Признаки:

| Признак | Описание |
|---|---|
| calls | количество звонков |
| minutes | суммарное количество минут разговора |
| messages | количество SMS |
| mb_used | объём интернет-трафика |
| is_ultra | целевая переменная (тариф) |

Размер датасета: **3214 наблюдений**.

---

## Используемые библиотеки

- pandas  
- scikit-learn  

Модели:
- Decision Tree
- Random Forest
- Logistic Regression

---

## Ход работы

1. Загрузка и изучение данных  
2. Разделение данных на выборки:
   - train — 56.25%
   - validation — 18.75%
   - test — 25%
3. Обучение нескольких моделей
4. Подбор гиперпараметров
5. Оценка моделей на тестовой выборке

---

## Результаты

Accuracy моделей на тестовой выборке:

| Модель | Accuracy |
|---|---|
| Decision Tree | 0.8047 |
| Random Forest | **0.8209** |
| Logistic Regression | 0.7550 |

Лучший результат показала модель **Random Forest**.

Дополнительные метрики (Random Forest):

- Precision: **0.8156**
- Recall: **0.4936**
- F1-score: **0.615**

---

## Итог

Модель **Random Forest** показала наилучшее качество классификации с точностью **0.8209**, что превышает требуемый порог **0.75**.  
Модель может быть использована для задачи рекомендации тарифов.
