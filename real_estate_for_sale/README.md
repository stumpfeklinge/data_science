# Анализ рынка недвижимости Санкт-Петербурга

## Описание проекта

Исследование архивных данных сервиса Яндекс.Недвижимость о продаже квартир в Санкт-Петербурге и соседних населённых пунктах за несколько лет. Цель проекта — определить параметры для построения автоматизированной системы оценки рыночной стоимости объектов недвижимости и выявления аномалий.

## Технологии

- **Python 3.8+**
- **Pandas** - обработка и анализ данных
- **Matplotlib/Seaborn** - визуализация
- **Jupyter Notebook** - интерактивный анализ
- **NumPy** - математические вычисления

## Данные
Датасет содержит 23 699 объявлений с 22 параметрами:

### Основные характеристики:
* total_images - количество фотографий

* last_price - цена квартиры, руб.

* total_area - общая площадь, м²

* first_day_exposition - дата публикации

* rooms - количество комнат

* ceiling_height - высота потолков, м

* floors_total - всего этажей в доме

* living_area - жилая площадь, м²

* floor - этаж квартиры

* is_apartment - является ли апартаментами

* studio - студия

* open_plan - свободная планировка

* kitchen_area - площадь кухни, м²

* balcony - количество балконов

### Локационные данные:
* locality_name - населённый пункт

* airports_nearest - расстояние до аэропорта, м

* cityCenters_nearest - расстояние до центра, м

* parks_around3000 - количество парков в радиусе 3 км

* parks_nearest - расстояние до ближайшего парка, м

* ponds_around3000 - количество водоёмов в радиусе 3 км

* ponds_nearest - расстояние до ближайшего водоёма, м

* days_exposition - количество дней продажи

## Методология исследования
### 1. Предобработка данных
* Преобразование типов данных

* Заполнение пропусков:

  * Медианные значения для числовых данных

  * Логические замены (0 для отсутствующих балконов)

  * Замена на большое значение для отсутствующих географических объектов

* Создание производных признаков

### 2. Исследовательский анализ
* Анализ распределений ключевых параметров

* Выявление и обработка выбросов

* Исследование временных характеристик продаж

### 3. Анализ факторов стоимости
* Корреляционный анализ

* Множественная регрессия

* Сравнительный анализ по локациям

### 4. Сегментационный анализ
* Анализ центральных районов Санкт-Петербурга

* Сравнение с пригородными зонами
